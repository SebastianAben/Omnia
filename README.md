# Omnia

SMART POS System for local Micro, Small, and Medium Enterprises.

## Monorepo Setup

Repository ini memakai `pnpm workspace` sebagai fondasi monorepo untuk Sprint 0.

## Struktur Awal

```text
apps/
  desktop-app/
  backend-api/
  ai-worker/
packages/
  ui/
  types/
  config/
  shared-utils/
docs/
```

## Package Manager

Gunakan `pnpm`.

## Root Scripts

- `pnpm lint`
- `pnpm format`
- `pnpm typecheck`
- `pnpm dev`
- `pnpm build`

## Catatan

- Setup ini hanya menyiapkan fondasi repo dan tooling, belum bootstrap penuh setiap aplikasi.
